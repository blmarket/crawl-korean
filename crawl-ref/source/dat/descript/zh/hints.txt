你的任务是扮演<w>{{ return you.race() }} {{ return you.class() }}</w>

directory

  <lightblue>quickstart.txt</lightblue>     - 简短的游戏入门说明。
  <lightblue>crawl_manual.txt</lightblue>   - 包括种族、魔法、
技能等等的详细说明。
  <lightblue>options_guide.txt</lightblue>  - 游戏设置详解。
Crawl的界面有很多地方是可以让玩家自由设置的。

祝你游戏愉快！

death

真遗憾！你角色的英年早逝令人悲伤，不过这在本作中是常有的事。不管怎么说，
失败是成功之母，你长久积累下来的游戏经验会帮助你以后的角色走得更远。

也许下面的一些建议可以帮助你提高游戏水平：
%%%%
death conjurer melee

作为一个塑能师，你的主要武器应是攻击性魔法。施法多多益善！牢记一点
：在魔力比较低的时候你需要休息来回复魔力。
%%%%

更强壮。

death ranger melee

你的弓与箭能非常有效地打击远处的敌人，所以应尽量搜集地城中有用的箭矢。
%%%%
death random 0

尽量在怪物近身之前使用弹药、魔杖或者法术打击它们。
%%%%
death random 1

学会在你搞不定的时候及时脱身——这非常重要
！跳过一些难度太大的楼层通常是明智之举。不过也不要过度使用这个方法，
因为随着地城的深入怪物会越来越强。
%%%%
death random 2

在打完一场战斗后要记得休息，而且最好是在一片已经被清扫完毕的区域。
游戏中搜索和休息是同时进行的。要搜索一回合，按下 <w>s</w>， <w>.</w>，
<w>delete</w> 或 <w>小键盘5</w>。按下 <w>5</w> 或 <w>shift+小键盘5</w>
<tiles>，或 <w>点击属性区域</w></tiles> 会让你休息更长时间
（你会停下来休息100回合，或是直到完全恢复为止）。
%%%%
death random 3

记得使用你找到的卷轴、药水和魔杖。通常来说，你不能依赖卷轴来鉴定所有的东西。
你必须学着应对这种情况；有时候你要通过亲自使用物品来鉴定它们。
%%%%
death random 4

在遇到难度太大的对手时记得早点使用应急物品。
一张传送卷轴或者一瓶加速药水往往可以救你一命。
%%%%
death random 5

除非别无选择，否则永远不要同时面对多个怪物。你应该退回到一个狭长通道，
这样怪物只能挨个与你战斗。

%%%%
finished explored

你可以使用自动探索机能 (<w>$cmd[CMD_EXPLORE]</w>) 来非常方便地探索地城。
游戏会将你自动移动至有物品的地方以便你拾取你感兴趣的东西。 <tiles>

你也可以在小地图上按下<w>Ctrl</w> + <w>鼠标左键</w> 来自动探索。</tiles>
%%%%
finished travel

你可以很方便地在各个楼层间旅行：按下 <w>$cmd[CMD_INTERLEVEL_TRAVEL]</w> 或
<w>G</w> 然后输入目的地便可。如果你的旅行被打断了，再按下
<w>$cmd[CMD_INTERLEVEL_TRAVEL] Enter</w> 或 <w>G Enter</w> 就可以继续。
%%%%
finished stashes

按下 <w>$cmd[CMD_SEARCH_STASHES]</w> 命令可以搜索地城中所有存在的物品。
例如：搜索 <w>$cmd[CMD_SEARCH_STASHES] "小刀"</w> 会显示出所有的小刀。
你可以选择其中的一个，然后自动移动至那里。你也可以搜索诸如 <w>"商店"</w> 或
<w>"祭坛"</w> 这样的关键词。
%%%%
finished random 0

游戏自动会对你的当前角色进行冒险记录。按下 <w>$cmd[CMD_DISPLAY_COMMANDS]:</w>
来查阅记录。你可以按下 <w>$cmd[CMD_MAKE_NOTE]</w> 输入你想记录的信息。
一旦你的角色死亡，在 <w>morgue/</w> 文件夹中会保留两个资料文档，其中那个 .txt
格式的文件就是该角色的游戏记录。在游戏中你可以按下
<w>$cmd[CMD_CHARACTER_DUMP]</w> 来输出一个缓存记录。
